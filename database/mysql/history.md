# MySQL 的歷史

## 1995

發布第一版 MySQL 的 My 是作者女兒的名字

## 2005

Oracle 是一家律師比工程師多的公司
而且律師的水準比工程師高

Oracle 買 Innobase Oy

## 2008

Sun 是主要賣硬體的公司

Sun 買 MySQL AB

## 2010

Oracle 買 Sun，承諾 5 年內不會縮減 MySQL 的營運

MariaDB 發布第一版，MariaDB 的 Maria 是同一個作者小女兒的名字

不想用 Oracle 的話可以找 Percona ， Percona 是技術底、專業做 MySQL 的公司
