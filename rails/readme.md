# 建立一個新專案

建立一個新專案的步驟：

- 確認 ruby 版號
- 確認 rails 版號
- rails new 生成專案
- Gem 初始設定
- 資料庫初始設定
- gitignore 初始設定
- rubocop 初始設定
- rspec 初始設定

# 確認 ruby 版號
在這個階段通常會選用最新的穩定版本 ruby
我使用 rvm 這個管理套件來管理 ruby 版本
